\# Student Course Management System 



A professional C++ Console Application for managing student records and course enrollments. 



\## Key Features

\- \*\*Persistent Storage\*\*: Saves data to `students\_db.txt` (Bonus Feature).

\- \*\*Data Integrity\*\*: Enforces Unique IDs and GPA range (0.0 - 4.0).

\- \*\*Course Management\*\*: Uses STL `set` to prevent duplicate enrollments.

\- \*\*Sorting\*\*: Ranks students by GPA using STL algorithms.



\## How to Run

\- \*\*Ensure all files (`main.cpp`, `Student.h`, `Student.cpp`, `StudentManager.h`, `StudentManager.cpp`) are in the same directory.

\- \*\*Compile using any C++ compiler (C++11 or later required).

\- \*\*Run the executable and follow the on-screen menu.





\## Applied Techniques

\- \*\*OOP\*\*: Inheritance (Student from Person), Encapsulation.

\- \*\*STL\*\*: Vector, Set, and Sort algorithms.

\- \*\*File I/O\*\*: Saving/Loading data from text files.
